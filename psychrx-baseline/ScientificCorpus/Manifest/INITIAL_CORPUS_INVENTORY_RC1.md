## Closure Criteria For A02.5-002

A02.5-002 can close only when:

- all official documents are identified;
- every document is registered in the corpus inventory;
- every document has a unique identifier;
- every document is organized in the `ScientificCorpus` structure;
- no scientific content has been interpreted or normalized.

## Declaration Final

The SSRI Source Corpus Initial Inventory RC1 defines the first official document set for the corpus. It does not validate, normalize, interpret or publish scientific knowledge.
