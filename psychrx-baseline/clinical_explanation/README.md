# Clinical Explanation Engine

The Clinical Explanation Engine transforms Runtime, Safety, Evidence, and Therapeutic Optimization artifacts into readable, traceable explanations.

It does not decide, recommend, prescribe, diagnose, or alter upstream results.

## Runtime Position

```text
Clinical Runtime
-> Safety
-> Evidence
-> Optimization
-> Explanation
-> Kernel
```

## Declaration

Clinical Explanation exists to make reasoning transparent to the physician. It explains the path; it does not choose the path.
