# Clinical Runtime

Execution coordination layer for PsychRx.

## Role

The Clinical Runtime coordinates sessions, context, pipeline, scheduler, events, store, validation, audit, trace, replay, and integrations.

## Hard Limits

- It does not make clinical decisions.
- It does not prescribe.
- It does not recommend therapies.
- It does not implement clinical reasoning engines.
- It does not validate real clinical conduct.
- It does not persist patient records.
