# Therapeutic Optimization Engine

The Therapeutic Optimization Engine is a structural, non-prescriptive reasoning engine.

It compares candidate therapeutic strategies as hypotheses for physician review. It does not prescribe, emit orders, choose medication, suggest doses, or replace medical judgment.

## Runtime Position

```text
Clinical Runtime
-> Safety Engine
-> Evidence Runtime
-> Therapeutic Optimization Engine
-> Clinical Kernel
```

## Current Status

- Structural package only.
- No medication-specific logic.
- No real therapeutic recommendation.
- No prescription.
- No automatic conduct.

## Declaration

Therapeutic Optimization supports clinical reasoning by producing traceable hypotheses, never autonomous treatment decisions.
