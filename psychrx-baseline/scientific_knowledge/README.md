# Scientific Knowledge

`scientific_knowledge/` contains the structural baseline for the Official Scientific Knowledge Base.

It defines schemas, contracts, package metadata, validation gates and editorial workflow for future scientific content.

It does not contain:

- clinical recommendations;
- populated psychopharmacology content;
- executable therapeutic rules;
- prescribing logic;
- AI inference.

Every future knowledge package must remain traceable, versioned, editorially reviewed and compatible with Scientific Validation Framework and Knowledge Governance Platform before it can be consumed by Evidence Runtime.
