# Knowledge Registry

## ID Prefixes

- `GLD` - guideline.
- `DRG` - psychopharmacological agent.
- `EVD` - evidence source.
- `INT` - interaction.
- `CTR` - contraindication.
- `ADV` - adverse effect.
- `MON` - monitoring.
- `REF` - reference.

## Rule

No registry entry is clinically official unless its validation status is `validated`.
