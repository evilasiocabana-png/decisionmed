# PsychRx Knowledge Library

Controlled scientific knowledge repository for PsychRx.

## Scope

This library stores structures, schemas, templates, registries, and draft-only examples.

## Hard Limits

- No validated clinical knowledge yet.
- No therapeutic recommendation.
- No prescription.
- No AI-generated clinical content.
- No feeding of clinical engines.
- No real psychopharmacological agent entry in this baseline.
