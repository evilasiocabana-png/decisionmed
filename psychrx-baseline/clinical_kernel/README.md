# Clinical Kernel

Structural package for future clinical reasoning infrastructure.

## Official Subpackages

- `context/`
- `state/`
- `pipeline/`
- `execution/`
- `registry/`
- `memory/`
- `audit/`
- `validation/`
- `errors/`

## Limits

This package is read-only infrastructure. It does not implement clinical engines, clinical decisions, recommendations, prescriptions, AI, evidence interpretation, or therapeutic rules.
