# Codex Execution Instructions

## Required Reading Order

1. `README.md`
2. `MISSION.md`
3. `CODEX.md`
4. `ACCEPTANCE.md`

## Execution Steps

1. Move package to `codex/processing/`.
2. Execute the mission.
3. Run validation.
4. Create `EXECUTION_REPORT.md` or `ERROR_REPORT.md`.
5. Move package to `codex/completed/` or `codex/failed/`.

## Safety

Do not execute outside the package scope.
