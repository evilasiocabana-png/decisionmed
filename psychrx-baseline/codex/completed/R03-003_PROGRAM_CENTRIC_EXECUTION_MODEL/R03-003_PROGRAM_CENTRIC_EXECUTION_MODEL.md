# MISSION R03-003 — PROGRAM_CENTRIC_EXECUTION_MODEL

## Objective
Adopt Program as the primary planning and execution unit of the PsychRx Project Operating System.

## Required Work
- Update governance to define Program as the user-facing execution unit.
- Define responsibilities:
  - User: approves Programs.
  - CTO: decomposes Programs into Phases, Sprints and Missions.
  - Codex: executes Tasks and Missions in sequence.
- Update PPOS documentation.
- Preserve backward compatibility.
- Generate an execution report.

## Constraints
- Governance only.
- Do not modify runtime.
- Do not modify scientific knowledge.
- Do not alter clinical logic.
