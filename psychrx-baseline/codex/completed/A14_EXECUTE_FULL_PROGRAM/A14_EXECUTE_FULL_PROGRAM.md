## Stop Conditions

Stop immediately and register a blocker if any of the following occur:

- failed gate;
- failed tests;
- missing scientific source material;
- incomplete traceability;
- invalid JSON or invalid state file;
- conflicting governance state;
- scope conflict;
- any attempted runtime enablement;
- any attempted prescribing or therapeutic recommendation.

## Permanent Prohibitions

The following remain prohibited throughout A14:

- clinical runtime eligibility;
- active runtime consumption;
- prescribing;
- therapeutic recommendations;
- dose suggestions;
- patient-specific medication selection;
- patient-facing clinical guidance;
- unsourced scientific claims.
