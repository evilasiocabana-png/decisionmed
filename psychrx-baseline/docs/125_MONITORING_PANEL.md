# 125 Monitoring Panel

## Objetivo

Definir o painel conceitual de monitorizacao clinica.

## Responsabilidades

- organizar acompanhamento;
- vincular monitorizacao a riscos e objetivos;
- acompanhar resposta clinica;
- registrar eventos adversos;
- apoiar deteccao de recaida, recorrencia, remissao e estabilidade.

## Regras

- monitorizacao deve ter motivo clinico;
- frequencia deve ser justificavel;
- eventos adversos relevantes devem alimentar o raciocinio longitudinal;
- ausencia de dados deve permanecer visivel.

## Declaracao Final

O Monitoring Panel conecta decisao, seguranca e evolucao longitudinal do paciente.
