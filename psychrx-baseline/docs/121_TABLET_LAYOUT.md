# 121 Tablet Layout

## Objetivo

Definir a adaptacao conceitual do dashboard medico para tablet.

## Principios

- priorizar leitura sequencial;
- reduzir densidade sem perder rastreabilidade;
- manter seguranca sempre acessivel;
- permitir navegacao fluida durante a consulta.

## Estrutura

- resumo do paciente;
- cards clinicos;
- perguntas pendentes;
- riscos;
- objetivos e restricoes;
- estrategias e evidencias;
- monitorizacao.

## Limites

Nao define tecnologia, biblioteca visual ou gestos especificos.

## Declaracao Final

O Tablet Layout traduz o raciocinio clinico para uma experiencia portatil sem remover explicabilidade nem seguranca.
