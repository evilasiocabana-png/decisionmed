# 426 - State Machine

## Objetivo

Definir uma maquina de estados explicita para o Clinical Operating Mind.

## Estados

- idle
- context_loaded
- safety_checked
- evidence_resolved
- hypotheses_generated
- explained
- snapshot_created
- timeline_updated
- navigation_ready
- blocked
- error

## Declaracao Final

A maquina de estados impede atalhos conceituais entre fases nucleares.
