# 285 - Evidence Metadata

## Objetivo

Padronizar metadados de evidencia.

## Campos

- title;
- authors;
- year;
- guideline;
- journal;
- organization;
- version;
- language;
- publication date;
- status;
- category.

## Declaracao Final

Evidence Metadata permite rastreabilidade bibliografica sem interpretar conteudo cientifico.
