# 339 - Explanation Language Guardrails

## Objetivo

Criar guardrails de linguagem para explicacoes clinicas.

## Proibido

- must prescribe;
- best medication;
- recommended prescription;
- definitive diagnosis;
- automatic treatment;
- dose recomendada;
- prescrever agora.

## Obrigatorio

- hypothesis;
- requires clinical judgment;
- based on available data;
- uncertainty when applicable.

## Declaracao Final

Language Guardrails impedem que explicacao vire prescricao disfarçada.
