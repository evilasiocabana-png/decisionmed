# 130 Offline Mode

## Objetivo

Definir limites conceituais para uso offline futuro.

## Permitido

- leitura de dados previamente sincronizados;
- consulta a snapshot local autorizado;
- registro temporario pendente de sincronizacao;
- exibicao clara de status offline.

## Proibido

- atualizar conhecimento cientifico sem validacao;
- ocultar que dados podem estar desatualizados;
- gerar conclusoes com evidencias indisponiveis;
- remover rastreabilidade.

## Declaracao Final

O Offline Mode deve preservar seguranca e transparencia quando conectividade, conhecimento ou sincronizacao estiverem limitados.
