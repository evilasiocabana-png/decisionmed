# 271 - Alert Formatting

## Objetivo

Criar formato de saida dos alertas para superficies read-only.

## Saidas

- Clinical Workspace;
- Logs;
- Developer mode;
- Audit.

## Limite

Formatacao nao altera severidade, nao decide conduta e nao muda contratos.

## Declaracao Final

Alert Formatting permite comunicacao consistente dos alertas sem alterar seu significado.
