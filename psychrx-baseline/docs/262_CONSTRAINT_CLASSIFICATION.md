# 262 - Constraint Classification

## Objetivo

Definir classificacao oficial de restricoes.

## Niveis

- `informational`;
- `caution`;
- `restriction`;
- `blocking`;
- `critical`.

## Uso

Esses niveis orientam contratos de severidade e priorizacao futura, sem decidir conduta clinica.

## Declaracao Final

A classificacao de restricoes padroniza linguagem de seguranca sem criar regras terapeuticas.
