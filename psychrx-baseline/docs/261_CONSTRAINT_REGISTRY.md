# 261 - Constraint Registry

## Objetivo

Criar o registro versionado de restricoes.

## Responsabilidades

- registrar restricoes;
- buscar por identificador;
- listar categorias;
- manter versao.

## Limite

O registry armazena definicoes estruturais e nao executa regras clinicas.

## Declaracao Final

O Constraint Registry preserva a separacao entre restricoes documentadas e execucao algoritimica.
