# 128 Mobile Card Stack

## Objetivo

Definir a pilha de cards clinicos para experiencia mobile.

## Cards Prioritarios

- alerta de seguranca;
- snapshot essencial;
- perguntas pendentes;
- objetivos;
- restricoes;
- monitorizacao;
- explicacao resumida.

## Regras

- cada card deve ser compreensivel isoladamente;
- riscos criticos nao podem ficar enterrados;
- acoes devem depender de revisao medica;
- conteudo deve preservar fonte e contexto.

## Declaracao Final

O Mobile Card Stack torna a experiencia mobile legivel sem sacrificar seguranca, contexto ou rastreabilidade.
