# 260 - Constraint Evaluator

## Objetivo

Criar o avaliador estrutural de restricoes.

## Responsabilidades

- receber contexto estruturado;
- avaliar apenas restricoes registradas;
- retornar restricoes, dados ausentes, severidade e confianca;
- preservar independencia entre conhecimento e algoritmo.

## Limite

Nenhuma restricao clinica real foi codificada.

## Declaracao Final

O Constraint Evaluator existe como ponto de extensao seguro para restricoes validadas futuras.
