# 444 - Failure Mode Tests

## Objetivo

Validar modos de falha estruturais.

## Casos cobertos

- transicao invalida;
- fase pulada;
- ADR ausente para mudanca estrutural;
- SafetyResult bloqueante;
- trace incompleto;
- saida proibida.

## Declaracao Final

As falhas sao tratadas como eventos estruturais, sem compensacao clinica automatica.
