# 132 Consultation Mode

## Objetivo

Definir o modo mobile de apoio durante consulta.

## Responsabilidades

- exibir resumo essencial;
- mostrar perguntas prioritarias;
- destacar riscos;
- registrar evolucao;
- apoiar explicacao ao paciente.

## Regras

- modo consulta deve reduzir distracao;
- informacoes devem ser ordenadas por relevancia clinica;
- medico deve controlar o ritmo;
- sistema nao deve pressionar por fechamento.

## Declaracao Final

O Consultation Mode apoia a consulta presencial ou remota sem substituir o encontro clinico.
