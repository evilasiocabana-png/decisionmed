# 163 - Widget Visibility Rules

## Objetivo

Definir regras visuais de visibilidade dos widgets.

## Estados de Visibilidade

`visible`, `collapsed`, `hidden`, `locked`.

## Limites

Aplicacao estatica nesta fase; nenhuma logica dinamica foi criada.

## Declaracao Final

As regras de visibilidade preparam o workspace para mostrar apenas o que importa no futuro.
