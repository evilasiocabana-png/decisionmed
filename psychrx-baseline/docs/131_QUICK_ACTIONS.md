# 131 Quick Actions

## Objetivo

Definir a politica conceitual para acoes rapidas no mobile.

## Acoes Permitidas Conceitualmente

- marcar dado como pendente;
- registrar observacao;
- abrir alerta;
- revisar evidencia;
- abrir monitorizacao;
- voltar ao snapshot.

## Acoes Proibidas

- prescrever;
- aprovar estrategia automaticamente;
- ocultar risco;
- pular Safety First;
- decidir conduta pela interface.

## Declaracao Final

Quick Actions devem acelerar acesso a informacao, nunca acelerar uma decisao clinica insegura.
