# 142 Audit API

## Objetivo

Definir conceitualmente a API de auditoria.

## Responsabilidades

- expor trilhas de decisao;
- registrar mudancas;
- rastrear fontes e versoes;
- preservar justificativas;
- permitir revisao institucional.

## Regras

- auditoria nao pode ser opcional em decisoes clinicas relevantes;
- eventos devem ter timestamp, origem e contexto;
- alteracoes de conhecimento devem manter versao;
- explicabilidade deve ser preservada.

## Declaracao Final

A Audit API garante que o PsychRx seja revisavel, rastreavel e responsavel.
