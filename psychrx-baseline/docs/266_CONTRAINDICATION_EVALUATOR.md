# 266 - Contraindication Evaluator

## Objetivo

Criar avaliador estrutural de contraindicoes.

## Responsabilidades

- aceitar contraindicoes futuras validadas;
- diferenciar contratos absoluto, relativo e condicional;
- preservar referencias futuras de evidencia.

## Limite

Nenhuma contraindicao clinica real foi inserida.

## Declaracao Final

O Contraindication Evaluator cria o ponto de extensao para contraindicoes sem codificar conteudo clinico.
