# 164 - Widget Documentation Standard

## Objetivo

Criar padrao oficial de documentacao de widgets.

## Cada Widget Deve Documentar

Nome, objetivo, contexto clinico, dados exibidos, dados proibidos, estados, permissoes, dependencias futuras e criterios de aceite.

## Declaracao Final

Este padrao reduz ambiguidade e retrabalho na evolucao dos Clinical Widgets.
