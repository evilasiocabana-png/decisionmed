# 425 - Lifecycle Definition

## Objetivo

Definir o ciclo oficial do Clinical Operating Mind.

## Fases

1. Context Intake
2. Safety Evaluation
3. Evidence Resolution
4. Therapeutic Hypothesis Generation
5. Explanation Assembly
6. Snapshot Creation
7. Timeline Update
8. Navigation Update

## Declaracao Final

O ciclo garante raciocinio antes de estrategia e seguranca antes de qualquer otimizacao.
