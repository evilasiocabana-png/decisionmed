# 139 Reasoning API

## Objetivo

Definir conceitualmente a API de raciocinio clinico.

## Responsabilidades

- expor estado do Clinical Operating Mind;
- apresentar incertezas;
- indicar perguntas pendentes;
- mostrar hipoteses, objetivos, restricoes e explicacoes.

## Regras

- raciocinio deve ser explicavel;
- nenhuma conclusao sem fonte quando houver conteudo cientifico;
- safety deve preceder estrategia;
- medico permanece decisor final.

## Declaracao Final

A Reasoning API torna o raciocinio clinico consultavel e auditavel sem transformar o sistema em prescritor.
