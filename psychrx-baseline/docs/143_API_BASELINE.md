# 143 API Baseline

## Objetivo

Consolidar a baseline conceitual da API Clinica.

## Componentes

- API Architecture;
- Case API;
- Patient API;
- Reasoning API;
- Strategy API;
- Monitoring API;
- Audit API.

## Criterios

- dominio independente;
- safety obrigatorio;
- rastreabilidade total;
- explicabilidade preservada;
- nenhuma prescricao automatica;
- medico decisor final.

## Declaracao Final

A API Baseline define a API Clinica como camada de acesso ao raciocinio do PsychRx, nunca como atalho para decisao automatica.
