# SSRI Scientific Scope

## Objective

Define the scientific boundaries of Program A03 before medication modeling.

## Covered

- editorial portfolio boundaries;
- source traceability requirements;
- knowledge domains required for each future SSRI profile;
- quality and publication gates;
- cross-reference standards.

## Not Covered

- dose guidance;
- indication guidance;
- comparative efficacy;
- adverse effect interpretation;
- interaction interpretation;
- contraindication interpretation;
- monitoring recommendations;
- prescribing.

## Editorial Criteria

Every future field must have:

- source reference;
- source version;
- traceability ID;
- review status;
- publication status.

## Declaration Final

Program A03 scope is structural and editorial at this phase.
