# 112 Clinical Workflow

## Objetivo

Definir o fluxo clinico conceitual da experiencia de uso do PsychRx durante uma consulta.

## Escopo

- organizar a sequencia da consulta;
- posicionar o paciente no centro;
- separar coleta, raciocinio, seguranca, explicacao e monitorizacao;
- preservar o medico como decisor final.

## Fluxo Conceitual

```text
Paciente
-> Clinical Snapshot
-> Perguntas necessarias
-> Hipoteses
-> Objetivos terapeuticos
-> Restricoes
-> Seguranca
-> Estrategias comparaveis
-> Explicacao
-> Monitorizacao
-> Seguimento longitudinal
```

## O Que Nao Fazer

- nao prescrever;
- nao substituir o julgamento medico;
- nao ocultar incertezas;
- nao apresentar estrategia sem justificativa;
- nao misturar interface com decisao clinica.

## Relacao Com Documentos

Depende de `008_CLINICAL_OPERATING_MIND.md`, `009_CLINICAL_SNAPSHOT.md`, `017_SAFETY_FIRST_ENGINE.md` e `030_CLINICAL_REASONING_ARCHITECTURE.md`.

## Declaracao Final

O Clinical Workflow define a experiencia de consulta como raciocinio clinico estruturado, seguro, explicavel e longitudinal.
