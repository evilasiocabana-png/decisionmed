# 161 - Widget Registry

## Objetivo

Criar o Widget Registry read-only.

## Widgets Registrados

Patient Header, Current Medication, Clinical Snapshot, Clinical Investigation, Objectives, Risk, Strategy, Monitoring, Clinical Compass, Missing Information e Consultation Progress.

## Limites

Nao cria runtime dinamico, banco, plugins ou IA.

## Declaracao Final

O Widget Registry e uma lista oficial read-only dos widgets do Clinical Workspace.
