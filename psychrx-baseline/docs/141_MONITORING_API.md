# 141 Monitoring API

## Objetivo

Definir conceitualmente a API de monitorizacao.

## Responsabilidades

- expor plano de acompanhamento;
- registrar resposta clinica;
- registrar eventos adversos;
- acompanhar estabilidade, recaida, recorrencia e remissao.

## Regras

- monitorizacao deve estar ligada a risco ou objetivo;
- eventos devem ser auditaveis;
- dados ausentes devem permanecer visiveis;
- API nao substitui seguimento medico.

## Declaracao Final

A Monitoring API apoia continuidade do cuidado e raciocinio longitudinal com rastreabilidade.
