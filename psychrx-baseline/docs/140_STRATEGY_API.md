# 140 Strategy API

## Objetivo

Definir conceitualmente a API de estrategias terapeuticas.

## Responsabilidades

- expor estrategias como alternativas;
- vincular estrategia a objetivo;
- exibir beneficio, risco, restricao, evidencia e explicacao;
- permitir comparacao controlada.

## Regras

- nenhuma estrategia sem Safety First;
- nenhuma estrategia sem justificativa;
- nenhuma estrategia sem rastreabilidade;
- API nao decide conduta.

## Declaracao Final

A Strategy API disponibiliza alternativas explicaveis para revisao medica, sem prescricao automatica.
