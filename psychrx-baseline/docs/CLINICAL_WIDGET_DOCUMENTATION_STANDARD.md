# Clinical Widget Documentation Standard

## Campos Obrigatorios

- Nome.
- Objetivo.
- Contexto clinico.
- Dados exibidos.
- Dados proibidos.
- Estados.
- Permissoes.
- Dependencias futuras.
- Criterios de aceite.

## Declaracao Final

Todo Clinical Widget deve ser documentado antes de ganhar comportamento dinamico.
