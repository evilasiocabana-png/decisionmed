# 127 Dashboard Baseline

## Objetivo

Consolidar a baseline conceitual do dashboard medico.

## Componentes Baseline

- Desktop Layout;
- Tablet Layout;
- Mobile Layout;
- Strategy Panel;
- Risk Panel;
- Monitoring Panel;
- Evidence Panel.

## Criterios

- nenhuma interface decide conduta;
- medico permanece decisor final;
- seguranca sempre visivel;
- estrategia sempre explicada;
- evidencia sempre rastreavel;
- incerteza sempre comunicada.

## Declaracao Final

A Dashboard Baseline define o dashboard medico como superficie de raciocinio, seguranca e explicabilidade, nao como prescritor.
