# 120 Desktop Layout

## Objetivo

Definir a organizacao conceitual do dashboard medico em desktop.

## Regioes Conceituais

- identificacao do paciente;
- Clinical Snapshot;
- perguntas pendentes;
- seguranca clinica;
- hipoteses;
- objetivos;
- restricoes;
- estrategias comparaveis;
- evidencias;
- monitorizacao;
- linha do tempo.

## Regras

- desktop deve favorecer comparacao e leitura profunda;
- seguranca deve permanecer visivel;
- interface nao decide conduta;
- evidencia e justificativa devem acompanhar toda sugestao.

## Declaracao Final

O Desktop Layout define a tela ampla como ambiente de raciocinio clinico comparativo, nao como painel prescritor.
