# 135 Mobile Baseline

## Objetivo

Consolidar a baseline conceitual da experiencia mobile.

## Componentes

- Mobile Card Stack;
- One Hand Navigation;
- Offline Mode;
- Quick Actions;
- Consultation Mode;
- Voice Mode;
- Responsive Validation.

## Criterios

- seguranca visivel;
- medico no controle;
- nenhuma prescricao automatica;
- incertezas explicitas;
- privacidade preservada;
- responsividade validada.

## Declaracao Final

A Mobile Baseline define uma experiencia clinica movel segura, assistiva e subordinada ao raciocinio medico.
