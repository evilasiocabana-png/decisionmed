# 133 Voice Mode

## Objetivo

Definir limites conceituais para um modo de voz futuro.

## Usos Possiveis

- registrar notas;
- resumir fala em rascunho;
- sugerir perguntas pendentes;
- recuperar informacoes do snapshot.

## Restrições

- voz nao deve prescrever;
- transcricao deve ser validada pelo medico;
- dados sensiveis exigem controle de privacidade;
- incerteza de transcricao deve ser visivel.

## Declaracao Final

O Voice Mode pode apoiar documentacao e navegacao, mas nao cria decisao clinica autonoma.
