# 129 One Hand Navigation

## Objetivo

Definir principios de navegacao mobile com uma mao para uso clinico rapido e seguro.

## Principios

- acesso rapido ao snapshot;
- acesso imediato a alertas;
- navegacao simples entre cards;
- retorno facil a contexto completo;
- evitar comandos irreversiveis.

## Regras

- nenhuma decisao clinica deve depender de gesto ambiguo;
- a interface deve reduzir erro, nao acelerar conduta;
- informacoes criticas exigem confirmacao visual clara.

## Declaracao Final

A One Hand Navigation melhora ergonomia sem transformar velocidade em criterio de decisao clinica.
