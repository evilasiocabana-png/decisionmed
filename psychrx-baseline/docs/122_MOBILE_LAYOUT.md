# 122 Mobile Layout

## Objetivo

Definir a experiencia conceitual do dashboard medico em telas pequenas.

## Prioridades

- leitura rapida;
- alertas de seguranca;
- snapshot essencial;
- perguntas prioritarias;
- timeline resumida;
- acesso a explicacoes.

## Regras

- mobile nao deve esconder riscos criticos;
- nao deve estimular decisao rapida sem contexto;
- nao deve substituir revisao completa em casos complexos;
- deve deixar claro quando ha incerteza.

## Declaracao Final

O Mobile Layout oferece continuidade e consulta rapida, sem reduzir a complexidade clinica a atalhos inseguros.
