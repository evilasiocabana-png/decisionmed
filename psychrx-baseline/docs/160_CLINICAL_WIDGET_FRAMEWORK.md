# 160 - Clinical Widget Framework

## Objetivo

Criar o framework estrutural dos Clinical Widgets.

## Contrato

`id`, `title`, `priority`, `state`, `context`, `dependencies`, `actions`, `explanation`, `visibility`, `permissions`.

## Limites

Nao cria framework externo, plugin, banco, IA ou runtime dinamico.

## Declaracao Final

O Clinical Widget Framework padroniza widgets como unidades de apresentacao clinica.
