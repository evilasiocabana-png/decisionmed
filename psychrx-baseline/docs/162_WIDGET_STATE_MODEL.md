# 162 - Widget State Model

## Objetivo

Definir estados oficiais dos Clinical Widgets.

## Estados

`ready`, `empty`, `pending`, `blocked`, `warning`, `critical`, `unavailable`, `future_integration`.

## Limites

Estados sao visuais e nao executam logica clinica.

## Declaracao Final

O Widget State Model padroniza estados sem criar raciocinio clinico.
