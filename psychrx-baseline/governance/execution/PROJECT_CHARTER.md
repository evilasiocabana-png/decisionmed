# Project Charter - PsychRx

## 1. Missao

PsychRx e uma plataforma de apoio ao raciocinio medico em psicofarmacologia.

Sua missao e organizar dados clinicos, conhecimento cientifico, riscos, estrategias, explicacoes e monitorizacao para apoiar o medico durante o raciocinio.

## 2. Problema que Resolve

A psicofarmacologia exige integrar sintomas, diagnosticos diferenciais, comorbidades, riscos, preferencias, evidencias, interacoes, efeitos adversos e evolucao longitudinal.

O PsychRx existe para reduzir dispersao cognitiva, melhorar rastreabilidade e apoiar decisoes clinicas responsaveis.

## 3. O que o PsychRx Nao E

O PsychRx nao e:

- prescritor automatico;
- substituto do medico;
- chatbot clinico autonomo;
- ferramenta de atendimento direto ao paciente;
- sistema de decisao sem supervisao;
- repositorio de opinioes sem evidencia.

## 4. Principios Inegociaveis

1. Paciente no centro.
2. Medico como decisor final.
3. Raciocinio antes de estrategia.
4. Seguranca clinica como primeira camada.
5. Nenhuma estrategia sem justificativa.
6. Nenhuma informacao clinica sem fonte rastreavel.
7. Conhecimento separado de algoritmo.
8. Interface nao decide conduta.
9. Explicabilidade obrigatoria.
10. Rastreabilidade total.

## 5. Governanca

O projeto e governado por:

- Constituicao Clinica;
- Manifesto;
- Principios Arquiteturais;
- Ontologia;
- ADRs;
- Master Development Plan;
- contratos de seguranca, evidencia e dependencia;
- Definition of Done;
- politica de testes.

## 6. Como o Projeto Evolui

O PsychRx evolui por:

```text
PROGRAM
-> PHASE
-> SPRINT
-> MISSION
-> TASK
```

Cada missao deve ser pequena, rastreavel, revisavel e segura.

## 7. Marcos Principais

| Marco | Resultado Esperado |
| --- | --- |
| Governanca | Regras, ADRs, DoD e politicas oficiais. |
| Arquitetura Cientifica | Modelos conceituais e separacao entre conhecimento e algoritmo. |
| Clinical Kernel | Nucleo computacional sem prescricao. |
| Clinical Experience | Experiencia de consulta organizada e assistiva. |
| Knowledge Population | Biblioteca cientifica versionada e rastreavel. |
| Clinical Reasoning | Motores clinicos explicaveis e testaveis. |
| Validation | Validacao cientifica, clinica e arquitetural. |
| Production | Operacao segura, auditavel e observavel. |

## 8. Definicao de Sucesso

O PsychRx sera bem-sucedido quando conseguir:

- apoiar raciocinio clinico sem substituir o medico;
- preservar seguranca clinica como primeira camada;
- tornar cada sugestao explicavel e rastreavel;
- manter conhecimento cientifico versionado;
- funcionar em consulta real sem atrapalhar o medico;
- permitir validacao e auditoria;
- evoluir sem caos arquitetural.

## Declaracao Final

O Project Charter define a identidade executiva do PsychRx e serve como porta de entrada para desenvolvedores, colaboradores cientificos, revisores e futuros parceiros.
