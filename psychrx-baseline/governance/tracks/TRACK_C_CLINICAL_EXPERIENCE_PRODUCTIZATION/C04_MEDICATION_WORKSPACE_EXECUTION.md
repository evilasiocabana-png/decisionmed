# C04 - Medication Workspace Execution

## Status

Executed as product and UX governance artifact.

## Scope

C04 defines a medication workspace for organizing current medication information.

## Result

The medication workspace may display medication name, dose as entered by the clinician, duration, adherence, adverse events and perceived response as documentation support.

## Restrictions Preserved

- No medication selection.
- No dose suggestion.
- No prescribing workflow.
- No recommendation to maintain, adjust, add or stop medication.
