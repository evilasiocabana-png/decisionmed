# C05 - Follow-up Workflow Execution

## Status

Executed as product and UX governance artifact.

## Scope

C05 defines the follow-up workflow around changes since the prior encounter.

## Result

The workflow organizes symptom change, stability change, treatment response, adherence, adverse events, functional status and monitoring needs.

## Restrictions Preserved

- No automated treatment change.
- No autonomous monitoring order.
- No clinical runtime decision.
