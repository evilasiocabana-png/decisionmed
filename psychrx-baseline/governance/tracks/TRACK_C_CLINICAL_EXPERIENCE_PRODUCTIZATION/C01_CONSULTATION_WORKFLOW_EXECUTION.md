# C01 - Consultation Workflow Execution

## Status

Executed as product and UX governance artifact.

## Scope

C01 defines the consultation workflow around the question:

```text
How is this patient today?
```

The flow prioritizes current clinical state, symptoms, observed signs, stability, current treatment, response, objectives, monitoring and follow-up.

## Result

The consultation workflow is productized as a non-prescriptive clinical organization layer.

## Restrictions Preserved

- No diagnosis automation.
- No prescription.
- No therapeutic recommendation.
- No clinical runtime activation.
