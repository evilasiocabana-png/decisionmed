# Directory Naming Standard - SSRI Knowledge Base

## Purpose

Define the naming rules for the Program A03 SSRI directory structure.

## Drug Directory Names

Drug root folders use canonical display names:

- `Fluoxetine`
- `Sertraline`
- `Paroxetine`
- `Citalopram`
- `Escitalopram`
- `Fluvoxamine`

## Section Directory Names

Section names use PascalCase without spaces:

- `Identification`
- `Mechanism`
- `Pharmacokinetics`
- `Pharmacodynamics`
- `Indications`
- `Posology`
- `Safety`
- `Contraindications`
- `Precautions`
- `Interactions`
- `AdverseEffects`
- `Monitoring`
- `Evidence`
- `Guidelines`
- `Editorial`
- `Metadata`
- `Traceability`
- `Manifest`

## File Names

Template file names use uppercase snake case where applicable:

- `README.md`
- `MANIFEST.json`
- `STATUS.json`
- `EDITORIAL_STATUS.json`
- `TRACEABILITY.json`
- `METADATA.json`

## Prohibited Naming

Do not create parallel synonyms such as:

- `DrugProfile`;
- `Drug`;
- `Medication`;
- `Recommendation`;
- `DoseRecommendation`;
- `ClinicalRule`.

These terms may be introduced only by future authorized missions and only if consistent with the official glossary and ontology.

## Declaration Final

Directory naming must preserve symmetry, traceability and governance. Naming consistency does not imply scientific validation.
