# A03 - Editorial Review Gate

## Objetivo

Definir o gate editorial minimo para populacao cientifica de SSRIs.

## Gate Obrigatorio

Cada medicamento deve passar por:

1. Source Intake
2. Field Extraction
3. Scientific Review
4. Semantic Review
5. Conflict Review
6. Editorial Decision
7. Version Assignment
8. Publication Gate

## Condicoes de Bloqueio

O pacote permanece bloqueado se houver:

- fonte ausente;
- fonte sem identificador;
- ano ausente;
- qualidade de evidencia ausente;
- conflito nao documentado;
- aplicabilidade clinica ausente;
- revisao editorial ausente;
- trace id ausente;
- tentativa de publicar regra terapeutica.

## Declaracao Final

O gate editorial impede que o PsychRx aceite conteudo cientifico sem curadoria formal.
