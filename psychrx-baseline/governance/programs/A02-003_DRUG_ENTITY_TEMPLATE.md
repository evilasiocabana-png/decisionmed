# A02-003 - Drug Entity Template

## Objetivo

Criar template canonico de entidade farmacologica.

## Secoes

Identity, classification, mechanism, receptors, indications, contraindications, interactions, adverse effects, monitoring, evidence, guidelines, ontology links, editorial review e traceability.

## Restricao

As secoes permanecem nao preenchidas no Program A02.

## Declaracao Final

O template de medicamento define lacunas obrigatorias, sem gerar conteudo clinico.
