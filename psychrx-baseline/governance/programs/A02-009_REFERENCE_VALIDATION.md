# A02-009 - Reference Validation

## Objetivo

Validar referencias cientificas cadastradas.

## Rejeicoes

- identificador ausente;
- referencia duplicada;
- fonte invalida;
- trace quebrado.

## Implementacao

Criado `ReferenceValidator`.

## Declaracao Final

Referencia invalida nao pode sustentar conhecimento cientifico futuro.
