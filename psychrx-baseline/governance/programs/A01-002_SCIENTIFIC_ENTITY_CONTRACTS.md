# A01-002 - Scientific Entity Contracts

## Objetivo

Definir contratos canonicos para entidades cientificas.

## Contrato Canonico

Cada entidade deve possuir:

- identifier;
- semantic id;
- scientific id;
- knowledge version;
- source references;
- evidence level;
- editorial status;
- trace id.

## Implementacao Estrutural

O contrato foi definido como `ScientificEntityContract`.

## Restricoes

O contrato nao descreve conduta, nao prescreve e nao valida evidencia real sozinho.

## Declaracao Final

Toda entidade cientifica futura devera nascer com identidade, versao, rastreabilidade e revisao editorial.
