# A02-001 - SSRI Package

## Objetivo

Criar o pacote oficial de SSRIs em modo estrutural.

## Estrutura

`scientific_knowledge/psychopharmacology/` passa a conter os contratos e catalogos iniciais para SSRIs.

## Status

Pacote registrado como `not_populated`.

## Restricoes

Nenhum dado cientifico, mecanismo, dose, indicacao, contraindicacao ou recomendacao foi adicionado.

## Declaracao Final

O pacote SSRI existe como estrutura versionavel e rastreavel, ainda vazio de conteudo cientifico.
