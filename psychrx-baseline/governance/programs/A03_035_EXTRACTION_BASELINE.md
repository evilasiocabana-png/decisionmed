# A03-035 - Extraction Baseline

## Program

Program A03 - Scientific Content Population: SSRIs.

## Phase

Phase 3.5 - Scientific Knowledge Acquisition.

## Mission

Freeze the first baseline of the Scientific Knowledge Acquisition layer.

## Baseline Scope

This baseline includes:

- A03-031 - Extraction Protocol;
- A03-032 - Extraction Schema;
- A03-033 - Normalization Rules;
- A03-034 - Scientific Review Workflow.

## Baseline Decision

Phase 3.5 is complete as governance for scientific acquisition.

## What This Baseline Allows Next

The project may now begin Phase 4 as controlled extraction and population of mechanism content, provided each mission:

- uses extraction records;
- binds every value to a source;
- preserves review workflow;
- avoids recommendation;
- avoids prescription;
- avoids runtime consumption.

## What Remains Prohibited

- clinical recommendation;
- prescription;
- automated decision;
- runtime consumption;
- unreviewed field publication;
- source-free scientific value.
