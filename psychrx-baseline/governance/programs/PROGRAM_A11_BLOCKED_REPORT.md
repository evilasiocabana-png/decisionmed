# Program A11 Blocked Report - Scientific Content Population: Second-Generation Antipsychotics

## Status

Bloqueado.

## Programa Solicitado

Track A - Scientific Knowledge Expansion

Program A11 - Scientific Content Population: Second-Generation Antipsychotics

- Risperidone;
- Paliperidone;
- Olanzapine;
- Quetiapine;
- Ziprasidone;
- Lurasidone;
- Asenapine;
- Iloperidone;
- Aripiprazole;
- Brexpiprazole;
- Cariprazine;

## Risco Evitado

- populacao de SGAs antes de validar pipeline cientifico;
- criacao de conteudo sobre agranulocitose, miocardite, riscos metabolicos, QT, prolactina e LAIs sem corpus versionado;
- falsa aprovacao editorial em classe de altissima criticidade;
- integracao prematura com Safety Engine, Evidence Runtime ou Clinical Operating Mind;
- risco de transformar conhecimento incompleto em alerta clinico ou conduta.

## Proxima Missao Autorizada

`MISSION A03-002 - SSRI Source Corpus Intake`

## Condicao para Desbloquear A11

O Program A11 so podera ser reaberto quando:

1. Program A03 tiver corpus, revisores e ao menos um piloto SSRI validado;
2. Programs A04-A10 estiverem desbloqueados ou houver ADR justificando excecao metodologica;
3. o pipeline de populacao, validacao, auditoria e publicacao estiver aprovado para antipsicoticos;
4. houver criterios especificos para clozapina, LAIs, risco metabolico, QT, prolactina, miocardite e agranulocitose;
5. houver decisao CTO autorizando expansao para SGAs.

## Declaracao Final

O Program A11 permanece bloqueado. A proxima execucao segura continua sendo A03-002.
