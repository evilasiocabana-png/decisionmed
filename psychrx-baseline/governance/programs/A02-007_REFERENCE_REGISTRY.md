# A02-007 - Reference Registry

## Objetivo

Criar registro de referencias cientificas.

## Tipos Suportados

- PMID;
- DOI;
- ISBN;
- Guideline;
- Book;
- Journal;
- Regulatory source.

## Implementacao

Criado `ReferenceRegistry`.

## Declaracao Final

Referencias sao registradas para rastreabilidade, nao interpretadas automaticamente.
