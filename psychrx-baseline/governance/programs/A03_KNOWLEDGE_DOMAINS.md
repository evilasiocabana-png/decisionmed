# A03 Knowledge Domains

## Domains

Each future SSRI knowledge object will be organized into these empty domains:

- Identification;
- Mechanism;
- Pharmacokinetics;
- Pharmacodynamics;
- Safety;
- Interactions;
- Contraindications;
- Monitoring;
- Evidence;
- Guidelines;
- Editorial;
- Traceability.

## Domain Boundary

Domains are containers for future validated content. They are not populated in Phase 1.

## Declaration Final

The Program A03 domain map defines where knowledge will live, not what the knowledge says.
