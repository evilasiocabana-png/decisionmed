# A03 Naming Standard

## ID Families

| Family | Pattern |
| --- | --- |
| Drug | `SSRI_<CANONICAL_NAME>` |
| Evidence | `A03_EVID_<NNNN>` |
| Mechanism | `A03_MECH_<NNNN>` |
| Safety | `A03_SAFE_<NNNN>` |
| Interaction | `A03_INT_<NNNN>` |
| Source Trace | `A02_5_SRC_<SOURCE_ID>` |

## Rules

- IDs must be stable.
- IDs must not encode clinical conclusions.
- IDs must not imply approval.

## Declaration Final

Naming is structural and does not create clinical assertions.
