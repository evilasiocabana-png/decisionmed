# A03 Cross Reference Standard

## Objective

Define how future A03 objects reference one another and their sources.

## Reference Types

- `source_reference`;
- `evidence_reference`;
- `guideline_reference`;
- `safety_reference`;
- `traceability_reference`;
- `editorial_reference`.

## Required Fields

- reference ID;
- source artifact;
- source version or edition;
- review status;
- publication status.

## Prohibited

- references without source;
- claims without traceability;
- runtime consumption before publication gate.

## Declaration Final

Cross references support future traceability. They do not create scientific claims.
