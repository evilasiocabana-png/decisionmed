# A04.0-001 - SNRI Source Discovery

## Program

PROGRAM A04.0 - SNRI Scientific Corpus.

## Phase

Phase 1 - Source Discovery.

## Mission

Identify candidate source families for the official SNRI Scientific Corpus.

## Objective

Create a raw discovery layer for SNRI source acquisition before any scientific modeling, interpretation, normalization or runtime use.

## Scope

The discovery scope covers the SNRI portfolio:

- venlafaxine;
- desvenlafaxine;
- duloxetine;
- levomilnacipran;
- milnacipran.
