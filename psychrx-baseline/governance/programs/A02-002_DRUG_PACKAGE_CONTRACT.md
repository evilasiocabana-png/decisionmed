# A02-002 - Drug Package Contract

## Objetivo

Definir contrato canonico para pacotes de psicofarmacos.

## Campos

- drug metadata;
- scientific references;
- knowledge version;
- editorial status;
- ontology mapping;
- validation status;
- publication status.

## Implementacao

Criado `DrugPackageContract`.

## Declaracao Final

Todo pacote farmacologico futuro deve ser governado por contrato explicito antes de receber conteudo.
