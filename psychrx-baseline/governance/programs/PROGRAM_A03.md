# Program A03 - SSRIs

## Status

Active in Phase 5 - Pharmacokinetics.

## Track

Track A - Scientific Knowledge Expansion.

## Objective

Build the official SSRI Knowledge Base from the published SSRI Scientific Source Corpus.

This program does not create software, artificial intelligence, autonomous clinical recommendations or prescription content.

## Current Gate

Program A03 is released for controlled SSRI mechanism extraction and normalization only.

Completed:

- Phase 1 structural foundation;
- Phase 2 editorial infrastructure;
- Phase 3 structural drug scientific modeling;
- Phase 3.5 scientific knowledge acquisition;
- A03-036A source anchor intake for mechanism extraction;
- A03-036 preliminary mechanism extraction for ready anchors;
- A03-037 mechanical normalization pending editorial review.
- A03-038 conflict and gap review;
- A03-039 partial mechanism baseline;
- A03-040 partial Phase 4 gate.

Program A03 remains blocked for:

- publication of mechanism fields without editorial approval;
- evidence grading;
- PK extraction before A03-041 source-anchor intake;
- PD assertions beyond their structural shells;
- safety assertions;
- clinical recommendation;
- prescription;
- runtime consumption.

## Expected Final Outputs

- SSRI Knowledge Base;
- Drug Registry;
- Evidence Registry;
- Mechanism Registry;
- PK Registry;
- PD Registry;
- Safety Registry;
- Interaction Registry;
- Contraindication Registry;
- Monitoring Registry;
- Editorial Registry;
- Traceability Registry;
- Publication Package.

## Phase 1 - Scientific Foundation

Phase 1 prepares the scientific and editorial foundation before any SSRI is modeled.

## Declaration Final

Program A03 starts as a governed scientific knowledge foundation. It does not populate clinical content or prescribe.
