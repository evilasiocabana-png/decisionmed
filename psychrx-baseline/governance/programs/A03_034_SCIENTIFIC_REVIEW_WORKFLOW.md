# A03-034 - Scientific Review Workflow

## Program

Program A03 - Scientific Content Population: SSRIs.

## Phase

Phase 3.5 - Scientific Knowledge Acquisition.

## Mission

Define the review workflow for extracted and normalized scientific claims.

## Review Stages

```text
Extraction Draft
-> Normalization Check
-> Source Traceability Check
-> Scientific Review
-> Conflict Review
-> Editorial Approval
-> Publication Gate
```

## Reviewer Responsibilities

Reviewers verify:

- source exists;
- source location is specific;
- field mapping is correct;
- extracted claim matches source;
- normalization did not change meaning;
- conflicts are marked;
- publication status is appropriate.

## Rejection Criteria

Reject extracted claims when:

- source is missing;
- source location is vague;
- field mapping is wrong;
- claim exceeds source;
- recommendation was introduced;
- prescribing instruction was introduced;
- conflict was hidden.
