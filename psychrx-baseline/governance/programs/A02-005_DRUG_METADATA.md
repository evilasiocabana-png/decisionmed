# A02-005 - Drug Metadata

## Objetivo

Definir metadados obrigatorios de medicamentos.

## Campos

- identifier;
- generic name;
- ATC;
- semantic id;
- knowledge id;
- editorial owner;
- review status;
- publication status.

## Implementacao

Criado `DrugMetadata`.

## Declaracao Final

Metadados identificam medicamentos, mas nao autorizam uso clinico.
