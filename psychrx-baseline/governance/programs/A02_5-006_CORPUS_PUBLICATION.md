## Publication Boundary

The publication is administrative and controlled. It allows the corpus to be referenced as a source package after Program A02.5 closure.

It does not authorize:

- SSRI scientific content population;
- Drug Profile creation;
- evidence classification;
- ontology mapping;
- therapeutic recommendation;
- prescription;
- Evidence Runtime consumption;
- Knowledge Layer publication of medication facts.
