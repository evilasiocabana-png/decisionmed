# A01-004 - Drug Knowledge Schema

## Objetivo

Definir o schema canonico para conhecimento de psicofarmacos sem inserir medicamentos.

## Escopo

O schema aceita referencias para:

- mecanismos;
- receptores;
- indicacoes;
- contraindicações;
- interacoes;
- efeitos adversos;
- monitorizacao.

## Implementacao Estrutural

Foi criado `DrugKnowledgeSchema`.

## Restricao

Nenhum psicofarmaco foi cadastrado.

## Declaracao Final

O modelo de psicofarmaco esta pronto para populacao futura, mas permanece vazio e sem regra terapeutica.
