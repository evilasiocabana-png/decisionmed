# Clinical Snapshot Engine

The Clinical Snapshot Engine consolidates runtime artifacts into a single immutable, versioned, traceable read model.

It is not a medical record, prescription, diagnosis, or clinical decision.

## Runtime Position

```text
Runtime -> Safety -> Evidence -> Optimization -> Explanation -> Snapshot
```

## Declaration

Clinical Snapshot is the computational state of reasoning at one point in time. It supports navigation, audit, comparison, and longitudinal evolution without replacing the physician or the official clinical record.
