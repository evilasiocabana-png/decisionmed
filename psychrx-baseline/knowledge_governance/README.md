# Knowledge Governance

Esta pasta organiza a governanca cientifica do PsychRx. Os documentos aqui definem como o conhecimento sera aceito, revisado, versionado e mantido.

Nao e um local para escrever conhecimento medico. E um local para regras, criterios e processos.

## Documentos

### 000_CONSTITUICAO_CIENTIFICA.md

Define a funcao cientifica central da base de conhecimento.

### 001_HIERARQUIA_DAS_FONTES.md

Define a ordem de prioridade entre tipos de fontes cientificas.

### 002_MATRIZ_DE_AQUISICAO_DO_CONHECIMENTO.md

Define como fontes futuras serao mapeadas para areas da base de conhecimento.

### 003_POLITICA_DE_ATUALIZACAO.md

Define como atualizacoes cientificas deverao ser acompanhadas e incorporadas.

### 004_POLITICA_DE_CONFLITO.md

Define como lidar com divergencias entre fontes.

### 005_NIVEIS_DE_EVIDENCIA.md

Define categorias e niveis de evidencia a serem usados pelo projeto.

### 006_CONTROLE_DE_QUALIDADE.md

Define criterios de qualidade para entrada e manutencao do conhecimento.

### 007_PROCESSO_DE_REVISAO.md

Define o fluxo de revisao antes de qualquer conhecimento ser considerado oficial.

### 008_VERSIONAMENTO_CIENTIFICO.md

Define como mudancas cientificas serao registradas ao longo do tempo.

### 009_FONTES_PRIORIZADAS.md

Define a lista futura de fontes prioritarias para aquisicao de conhecimento.

### STRUCTURE_REPORT.md

Registra a criacao inicial da estrutura de governanca cientifica.
